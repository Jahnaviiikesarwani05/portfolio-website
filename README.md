# 🌐 Personal Portfolio Website

A responsive personal portfolio website built using **HTML** and **CSS**. This project showcases my profile, skills, projects, education, and contact information through a clean and modern user interface..


---

## 📌 Features

- 🏠 Home section with introduction
- 👤 About Me section
- 💻 Skills section
- 📂 Projects showcase
- 🎓 Education details
- 📞 Contact section
- 📱 Responsive design for different screen sizes
- 🎨 Clean and modern layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

```
Portfolio/

```

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

2. Open the project folder.

3. Double-click `index.html` or open it in any modern web browser.

No installation or additional software is required.
